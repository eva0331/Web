# 项目源码说明:
	waimai  客户端应用
	waimai-serve  服务器端应用


# 项目运行说明
## 1. 准备
	1) 确保安装了node环境
		查看是否已经安装: node -v

	2) 安装node_modules
		执行命令：npm i

	3) 确保安装了mongodb, 并启动了数据库服务
		管理员身份打开 powershell 
		查看是否已经安装: mongod --version
		启动数据库服务：net start mongodb
		连接数据库：mongo
		断开连接：exit
	
## 2. 启动后台应用
	1). 进入waimai-serve
	2). 执行命令: npm run start

## 3. 启动前台应用并访问
	1). 进入waimai
	2). 执行命令: npm run build
